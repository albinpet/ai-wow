# Quick Start Guide

Get up and running with AI-powered development in 10 minutes!

## ⚡ 5-Minute Setup

### 1. Install Cursor (2 minutes)
```bash
# Download and install Cursor from cursor.sh
# Or use package managers:

# macOS
brew install --cask cursor

# Windows (if available via winget)
winget install Cursor
```

### 2. Clone This Repository (1 minute)
```bash
git clone https://github.com/albinpet/ai-wow.git
cd ai-wow
cursor .
```

### 3. Try Your First AI Interaction (2 minutes)
1. Open Cursor and press `Cmd/Ctrl + K`
2. Ask: "Explain what this repository is about"
3. Try: "Show me how to create a simple Python function with AI"

## 🎮 Try These Examples Right Now

### Example 1: AI Code Generation
1. Create a new file: `my-first-ai-code.py`
2. Press `Cmd/Ctrl + K` in Cursor
3. Type: "Create a simple calculator class with add, subtract, multiply, and divide methods"
4. Watch AI generate the code!

### Example 2: Code Review with Claude
1. Copy the generated calculator code
2. Go to [claude.ai](https://claude.ai)
3. Paste this prompt:
   ```
   Review this Python code for best practices and suggest improvements:
   
   [paste your calculator code here]
   ```

### Example 3: Documentation Generation
1. Back in Cursor, select your calculator code
2. Press `Cmd/Ctrl + K`
3. Ask: "Generate comprehensive documentation for this code including docstrings and usage examples"
