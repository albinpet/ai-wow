# AI WOW - AI-Powered Development Workflow

> Demonstrating effective ways of working with AI tools using Cursor, Claude, and MCP GitHub integration

## 🚀 Getting Started

This repository showcases modern AI-assisted development workflows, demonstrating how to effectively integrate and use various AI tools to enhance productivity and code quality.

### ⚡ Quick Start
**Want to jump right in?** Check out our [Quick Start Guide](./QUICKSTART.md) to get up and running in 10 minutes!

### Prerequisites

Before you begin, ensure you have the following tools installed:

- **Cursor Editor** - AI-powered code editor
- **Claude** - Anthropic's AI assistant (via API or web interface)
- **Git** - Version control
- **GitHub CLI** (optional) - For enhanced GitHub integration

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/albinpet/ai-wow.git
   cd ai-wow
   ```

2. **Open in Cursor**
   ```bash
   cursor .
   ```

3. **Explore the examples**
   - Browse the `/examples` directory for workflow demonstrations
   - Check out the `/docs` folder for detailed guides
   - Review configuration files for AI tool setups

## 🔧 AI Tools Integration

### Cursor Setup
- Install Cursor from [cursor.sh](https://cursor.sh)
- Configure your AI model preferences
- Set up custom rules and prompts

### Claude Integration
- Access Claude via [claude.ai](https://claude.ai)
- Use Claude for code review and documentation
- Leverage Claude's reasoning capabilities for complex problem solving

### MCP GitHub Integration
- Connect GitHub with Model Context Protocol
- Enable seamless repository interactions
- Automate issue management and PR workflows
